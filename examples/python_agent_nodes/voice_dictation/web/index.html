🎙️ AgentField Voice Dictation

Wispr-Flow-style live transcription, routed through an AgentField realtime session. Mic → WebRTC → control-plane SDP proxy → OpenAI Realtime. Your provider key never leaves the server.

idle

The browser POSTs its SDP offer to /api/v1/sessions/<id>/realtime-offer?provider=openai&transport=webrtc; the control plane forwards it to OpenAI with its server key and returns the answer SDP. Once connected, the page sends a session.update over the data channel to enable input transcription and disable model replies (pure dictation) — a client-side stand-in until session-level VAD/transcription config lands (issue #663).
